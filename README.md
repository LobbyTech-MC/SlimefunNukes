# SlimefunNukes(自定义粘液核弹) ☢

#### 想在Slimefun中添加点艺术色彩？这个插件就是给你的

SlimefunNukes 允许你使用Slimefun上现有的物品制作、放置和引爆各种武器和爆炸物。

## 下载 🚀 

_构建条件:Java开发工具包11或更高版本和Maven_

1. 克隆存储库并访问目录
```sh
git clone https://github.com/aurgiyalgo/SlimefunNukes
cd SlimefunNukes/
```

2. 用Maven构建项目

```sh
mvn package
```
